Download MPExtended source

Our source code is hosted in Git on GitHub. Please see our repository page there for more information and the source code for the latest development version, which can be cloned with Git from git://github.com/MPExtended/MPExtended.git.

The source code of the latest stable version, 0.7.13, can be found in the repository under the tag v0.7.13, or be downloaded directly in release.
