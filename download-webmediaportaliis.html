WebMediaPortal IIS Edition

This is an edition of WebMediaPortal for people who already have the standalone Internet Information Services (IIS) webserver installed and want to use it for WebMediaPortal. Note that this is a different product than the IIS Express webserver, which is used for the regular version of WebMediaPortal. You have to manually configure IIS for this edition. If you want an easy setup, don't have IIS or want to use IIS Express, please use the standalone edition.
